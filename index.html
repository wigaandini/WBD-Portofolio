Pattern Matching App

The Pattern Matching in Biometric Fingerprint Detection System is an application for fingerprint authentication. Developed with WPF C# and MySQL, it uses Boyer-Moore and Knuth-Morris-Pratt algorithms for 100% fingerprint matches. If no exact match is found, the Levenshtein algorithm calculates the similarity percentage, ensuring accurate identification.

Large-scale Network

The project "Optimizing Shortest Path Calculation on Large-Scale Road Networks Using the A* Algorithm" focuses on enhancing shortest path calculations for large road networks. Implemented in Python, it leverages frameworks like NetworkX, OSMnx (real map), and GeoPandas (geographic data). These tools efficiently calculate optimal routes on large-scale road networks.
